# Normals Preview Highlights

The preview toolbar provides five local display modes.

- **Fast** marks the lowest startup normal in each card. Tied startup values are all marked.
- **Damage** marks the largest displayed primary-hit damage value in each card.
- **+Hit** marks the best calculated hit advantage in each card.
- **+Block** marks the best calculated block advantage in each card.
- Clicking an active mode clears that mode.

## Punish mode

1. Click the source normal in either team card.
2. Click **Punish**.
3. Cards on the opposing team mark the highest-damage normal that starts within the selected normal's negative block-advantage window.

The source row uses an amber outline. Highlighted results use a subtle colored rail and underline. If the selected normal is neutral or positive on block, the toolbar reports that it is safe on block; no standard normal punish is marked.
