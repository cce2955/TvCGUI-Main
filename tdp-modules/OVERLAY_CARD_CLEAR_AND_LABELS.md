# Overlay Card Clear + Full Collision Labels

## Dock changes

- The collision master controls now read `Hitboxes: ON/OFF` and `Hurtboxes: ON/OFF`.
- Added `Clear` beside `Hit/Blk`, `Combo`, and `Tag`.
- `Clear` disables those three optional live cards in one click.
- `Clear` does not disable the main Overlay HUD or collision displays.

## Ruler behavior

- A fresh Hitboxes OFF -> ON action still turns on the ground-normal ruler by default.
- No range profile data is modified by this build.
