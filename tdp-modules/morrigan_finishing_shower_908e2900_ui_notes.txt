Morrigan Finishing Shower UI patch notes

User-confirmed live missile record:
- record base: 0x908E2900
- damage:     0x908E2906, u16, confirmed 0x0320 / 800
- radius:     0x908E2930, f32, confirmed reactive
- origin:     0x908E295F, u8, confirmed spawn/origin/bone-ish selector
- speed:      0x908E2990, f32, user-adjusted from earlier wrong 0x908E2994

What this patch does:
- Adds a scanner path for the custom Morrigan Finishing Shower missile block.
- Adds a dedicated projectile row format: morrigan_fs_missile.
- Suppresses the old 00/23 Finishing Shower Bullet rows once the real 800-damage missile row is found.
- Adds a Spawn Origin projectile UI field.
- Routes edits for the Morrigan missile row only to:
  - Damage       = base + 0x06
  - Radius       = base + 0x30
  - Spawn Origin = base + 0x5F
  - Speed        = base + 0x90
- Prevents this custom row from falling back to generic template offsets, so +0x5B is not mislabeled as lifetime.

Expected visible UI:
- Finishing Shower Emitter should summarize 1 confirmed missile template, not the old 48..112 card-list rows.
- Finishing Shower Missile should show damage 800, radius 1, origin 5, speed 100 when unchanged.
