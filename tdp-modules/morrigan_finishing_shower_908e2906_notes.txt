Morrigan Finishing Shower: confirmed 800-damage local template

Confirmed by user:
  Per-missile damage address: 0x908E2906
  Damage value: 800 decimal = 0x0320

Record base:
  0x908E2904 = 0x908E2906 - 0x02

This is not a 00/23 compact-card damage field. It is the normal projectile-template layout:
  base + 0x02 = Damage u16
  base + 0x04 = 00 00 00 0C suffix
  base + 0x08 = discriminator
  base + 0x24 = KB X f32
  base + 0x28 = KB Y f32
  base + 0x2C = Radius f32
  base + 0x42 = validation u16, usually 10
  base + 0x51 = projectile type u8
  base + 0x52 = projectile ID u16
  base + 0x5B = lifetime u8
  base + 0x6E = hitbox size u16
  base + 0x80 = speed f32
  base + 0x84 = accel f32
  base + 0x8C = hitbox f32
  base + 0x90 = arc f32
  base + 0x94 = arc2 f32

Immediate poke ladder:

1) Damage
  0x908E2906 expected: 03 20
  Test high: 03 20 -> 0C 80  (800 -> 3200)
  Test low:  03 20 -> 00 64  (800 -> 100)
